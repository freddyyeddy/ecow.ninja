to search for children of a PArent number "where wonum in (select wonum from woancestor where ancestor = '1839207')"

If you run a report and dont email it (just click run with the immidiate box checked) then add "__format=pdf" to the end of the url of the new tab that opened you will see a pdf of the workorder you opened (Maximos BIRT report tool doesnt support explorer-11 go figure)

If you want to only search for whats in stock at c pad paste this into a where query
"(inventoryid in (select inventory.inventoryid from inventory inner join invbalances on inventory.siteid = invbalances.siteid and inventory.itemnum = invbalances.itemnum and inventory.itemsetid = invbalances.itemsetid and inventory.location = invbalances.location and invbalances.curbal >1)) and ((location like '%NS01%')) "
Then save the query as default then filter the stocked items from there
